# **App Name**: TaskTrack

## Core Features:

- Task Creation: Allow users to create new tasks with descriptions and due dates.
- Task Listing: Display all tasks with sorting and filtering options.
- Task Editing: Enable users to edit existing tasks.
- Task Completion: Allow marking tasks as complete.
- Task Persistence: Store tasks in a Firestore database to allow persisting tasks accross sessions and devices.

## Style Guidelines:

- Primary color: Cool blue (#64B5F6) for a calm and organized feel.
- Background color: Light gray (#F5F5F5), almost white but with a hint of coolness, provides good contrast and is easy on the eyes.
- Accent color: Soft green (#81C784) to indicate completion or success.
- Body and headline font: 'Inter', sans-serif, providing a modern and clean appearance suitable for a task management app.
- Use clear, minimalistic icons for task status and actions.
- Clean, card-based layout for easy readability and task management.
- Smooth transitions and animations when tasks are added, edited, or completed.
